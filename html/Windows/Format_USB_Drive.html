format a USB drive

A simple batch script that you can use to format a USB drive in Windows:

@echo off

rem Set the drive letter of the USB drive (Change G for you own drive letter)
set USB_DRIVE=G:
    
rem Format the USB drive (Change the name MY_USB)
format %USB_DRIVE% /fs:ntfs /v:MY_USB
    
echo Done!
To use this script:
Save it to a file with a format-usb.bat extension.
Then, open a Command Prompt window, navigate to the directory where the script is saved, and run the script by entering the name of the file or double click.

This will format the USB drive with the drive letter G: using the NTFS file system, and give it the volume label "MY_USB".

Note: This script will erase all data on the USB drive, so make sure to back up any important files before running the script.

reboot-and-shutdown

A batch script that will reboot Windows and then shut down after the reboot:

@echo off

rem Reboot the system
shutdown /r /t 0

rem Wait for the system to come back online
timeout /t 60

rem Shut down the system
shutdown /s /t 0

To use this script, save it to a file "reboot-and-shutdown.bat" and double-click it to run it.

This script will reboot the system and then shut it down after a delay of 60 seconds to allow the system to come back online. You can adjust the delay by changing the value of the timeout command.

Note: You will need to have administrative privileges to run this script.

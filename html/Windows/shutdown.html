Shortcuts


To create a shortcut for shutting down, rebooting, or putting your computer into hibernation in Windows, follow these steps:
    First things first
  1. Right-click on your desktop and select "New" followed by "Shortcut".
  2. In the "Create Shortcut" window, enter the following command for the action you want to perform:
    Choose one of these options
  • Shutdown: shutdown -s -t 00
  • Reboot: shutdown -r -t 00
  • Hibernate: rundll32.exe PowrProf.dll,SetSuspendState
    And now the final steps
  1. Click the "Next" button.
  2. In the "Type a name for this shortcut" field, enter a name for the shortcut.
  3. Click the "Finish" button.
You should now have a new shortcut on your desktop that you can double-click to perform the selected action.


Note:
The -s flag specifies that the action should be a shutdown, the -r flag specifies that the action should be a reboot, and the -t 00 flag specifies that the action should be executed immediately. The rundll32.exe command is used to execute a function in a dynamic-link library (DLL). In this case, it is used to execute the SetSuspendState function in the PowrProf.dll library, which puts the computer into hibernation.

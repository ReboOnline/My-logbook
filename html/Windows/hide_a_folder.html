To hide a directory in Windows using the Command Prompt, follow these steps:
  1. Open the Command Prompt by typing "cmd" into the Start menu search bar and pressing Enter.
  2. Navigate to the directory that you want to hide. You can do this by using the cd command followed by the path of the directory. For example, if the directory is located in the "C:\Users\Username" directory, you can use the command cd C:\Users\Username.
  3. Type the following command and press Enter: attrib +h [directory name]
Replace [directory name] with the name of the directory you want to hide. For example, if the directory is named "hidden_directory", you would use the command attrib +h hidden_directory.

This will add the hidden attribute to the directory, which will make it hidden from view in Windows Explorer and other file browsers. To view hidden directories, you will need to show hidden files and folders in your file browser's settings.

To unhide the directory, use the same command, but replace the +h with a -h. For example, attrib -h hidden will remove the hidden attribute from the "hidden" directory.

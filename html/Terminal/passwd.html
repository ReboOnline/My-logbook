Change your password in Linux via the terminal

Open a terminal.
Type in the passwd command
Type in the terminal your current password and press Enter.
Type in the passwd command
Type in your new password in the terminal, press Enter.
Then type it in again to confirm that it is correct.
Note that Linux will not echo (print) the password to the screen, nor will it show any asterisk indicating password length.

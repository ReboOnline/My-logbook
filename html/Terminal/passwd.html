Change your password in Linux via the terminal

Open a terminal of choice.

Type in the passwd command

Type in your new password in the terminal, press Enter.

Then type it in again to confirm that it is correct.
Note that Linux will not echo (print) the password to the screen, nor will it show any asterisk indicating password length. Type in the terminal your current password and press Enter.

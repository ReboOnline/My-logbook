Linux System Optimizer & Monitoring

Stacer is an open source system optimizer and application monitor that helps users to manage entire system with different aspects, its an all in one system utility.

For Ubuntu like distro's

Open an terminal and type in:

  1. sudo add-apt-repository ppa:oguzhaninan/stacer
  2. sudo apt-get update
  3. sudo apt-get install stacer


For Arco Linux

Open an terminal and type in:

  1. yay stacer
  2. choose optie 2 and click enter
  3. Follow the insallation and enjoy your program

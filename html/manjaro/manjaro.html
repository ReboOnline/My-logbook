6 Essential Things To Do After Installing Manjaro Linux

1. Set the fastest mirror

Before even updating the system, I suggest to sort out your mirror list first.
When refreshing the Manjaro system and downloading software from repositories,
an optimized mirror list can have noticeable performance impact to the system.
sudo pacman-mirrors --fasttrack

Update Manjaro

Type in de terminal sudo pacman -Syu

Force update Manjaro

Wanneer de update een error geeft in een pakket kun je sudo pacman -Syyu gebruiken.
De extra y dwingt de pakketbeheerder om de pakketdatabase te downloaden, ongeacht of er een wijziging in de versies is of niet.
Dit is handig wanneer u een beschadigde pakketdatabase hebt en u een synchronisatie wilt forceren.
Ik raad niet aan om sudo pacman -Syyu altijd te gebruiken, omdat het onnodig veel tijd kost om de database te synchroniseren.

Install Software in Manjaro Linux with pacman

Type in de terminal sudo pacman -S PACKAGENAME
Just replace PACKAGENAME with the name of the package you want to remove.

Remove Software in Manjaro Linux with pacman

Type in de terminal sudo pacman -R PACKAGENAME
Just replace PACKAGENAME with the name of the package you want to remove.

Install and Remove Software in Manjaro Linux with yay

Install yay on your machine to install packages from both official repository and AUR.
To do this, type in the commands below:
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

One nice thing about yay is that it gives you the ability to search for the application you want to install, all you have to do is use this command:
yay -S PACKAGENAME
Just replace PACKAGENAME with the name of the package you want to remove.

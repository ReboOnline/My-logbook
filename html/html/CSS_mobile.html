In the example ablow, we have a grid layout with three columns defined using grid-template-columns: 1fr 1fr 1fr;.
Each grid item has a background color and some padding for styling purposes.

Apply media queries to adjust the number of columns based on the device size:

    For mobile devices with a maximum width of 767 pixels, we use grid-template-columns: 1fr; to change the layout to a single column.
    For iPads with a width between 768 and 1023 pixels, we use grid-template-columns: 1fr 1fr; to change the layout to two columns.
    For larger screens with a minimum width of 1024 pixels, we use grid-template-columns: 1fr 1fr 1fr; to keep the original three-column layout.

Customize the grid layout, the number of columns, and the styles within each media query block to suit your specific design requirements.

HTML <div class="grid-container"> <div class="grid-item">Item 1</div> <div class="grid-item">Item 2</div> <div class="grid-item">Item 3</div> </div>

CSS .grid-container { display: grid; grid-template-columns: 1fr 1fr 1fr; grid-gap: 10px; } .grid-item { background-color: #ccc; padding: 20px; text-align: center; } /* Media query for mobile devices */ @media (max-width: 767px) { .grid-container { grid-template-columns: 1fr; } } /* Media query for iPads */ @media (min-width: 768px) and (max-width: 1023px) { .grid-container { grid-template-columns: 1fr 1fr; } } /* Media query for larger screens */ @media (min-width: 1024px) { .grid-container { grid-template-columns: 1fr 1fr 1fr; } }

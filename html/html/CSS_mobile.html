De lege CSS
/* Styles for mobile devices */ @media (max-width: 767px) { /* CSS rules for mobile devices (less than or equal to 767px) */ }

/* Styles for iPads */ @media (min-width: 768px) and (max-width: 1023px) { /* CSS rules for iPads (between 768px and 1023px) */ }

/* Styles for larger screens */ @media (min-width: 1024px) { /* CSS rules for larger screens (greater than or equal to 1024px) */ }

In het onderstaande voorbeeld is de eerste mediaquery gericht op apparaten met een maximale breedte van 767 pixels, wat doorgaans overeenkomt met mobiele apparaten.
De tweede mediaquery is gericht op apparaten met een minimale breedte van 768 pixels en een maximale breedte van 1023 pixels, waaronder over het algemeen iPads en andere tablets.
De derde mediaquery stijl heeft een minimale breedte van 1024 pixels, voor grotere schermen zoals desktops of laptops.

In onderstaand voorbeeld heb ik je een "grid-layout" met drie kolommen gemaakt met behulp van grid-template-columns: 1fr 1fr 1fr;. Elk grid item heeft een achtergrondkleur en wat opvulling voor stylingdoeleinden.
Nu kun je voor alle mediaquery's op basis van de apparaat grootte de lay-out aanpassen:

Voor mobiele apparaten met een maximale breedte van 767 pixels gebruik je grid-template-columns: 1fr; om de lay-out te wijzigen in een enkele kolom.
Voor iPads met een breedte tussen 768 en 1023 pixels gebruik je grid-template-columns: 1fr 1fr; om de lay-out te wijzigen in twee kolommen.
Voor grotere schermen met een minimale breedte van 1024 pixels gebruik je grid-template-columns: 1fr 1fr 1fr; om de originele lay-out met drie kolommen te behouden.

Lists for HTML
In addition to the usual ordered lists <ol> and unordered lists <ul> , there are a few other types of lists you can use in HTML.

   The ordered list:
  1. Use <ol> for a numbered list.
  2. Items are numbered by default (1, 2, 3, ...).
  3. Use the type attribute to change the numbering style (e.g., type="A" for uppercase letters, type="a" for lowercase letters, etc.).
  4. Use the start attribute to start the counter from a specific number.

   The unordered list:
  • Use <ul> for a unordered list.
  • Items are marked by bullitpoint (unordered).
  • Use the type attribute to change the numbering style (e.g., type="-" for uppercase letters, type="a" for lowercase letters, etc.).
  • Use the start attribute to start the counter from a specific number.

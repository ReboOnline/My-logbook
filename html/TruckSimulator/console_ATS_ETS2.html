ATS & ETS2 | Cheat-codes


This guide was created to make it easier for me to find cheat codes.
Here I collected almost all the codes that might come in handy.


Turning on the console

First, to start the console, you need to go to this directory:
My Computer\Documents\American Truck Simulator

NOTE: Or to the folder where you installed your truck simulator if you changed it yourself.


Find the"config.cfg"

Find the "config.cfg" text pad and open it.
Then find the values "uset g_developer" and "uset g_console"
Next, change the value from "0" to "1" - this will enable developer mode and console.

Backup your profile

You can copy your profile in both ATS and ETS2.
This can be useful as a backup when you want to test with new mods.
Or if you want multiple profiles and don't want to start all over again.




Step 1
  1. Choose the profile you want to backup
  2. Click on "Edite Profile"




Step 2
  1. Uncheck "Use Steam Profile"




Step 3
  1. Click "OK" and then "Apply"
  2. Start the profile and save it, now it will be in the local save folder
  3. Shut down ATS or ETS2 completely




Step 4
  1. Go in to your local save folder (Example: Documenten\American Truck Simulator\profiles\)
  2. Copy and paste the last saved profile to the same folder, so you will have two folders with te same name and the new one ends with "copy"
  3. download the SII_Decript here ---> https://sharemods.com/h6pozzt4ti8h/SII_Decrypt.rar.html
  4. Extract the decrypter and place it in the local profile folder
(NOTE: There maybe just one for you)




Step 5
  1. Drag the "profile_sii.exe" onto the SII_Decript and click yes if necessary
  2. Remove the "profile_sii.exe" from the folder




Step 6
  1. Open the profile.sii with notepad or notepad++
  2. Scroll all the way down to the line with "profile_name" and change only the old profile name, for example: profile_name YourNewName
  3. Copy the new name for the next step
Note:The name cannot have spaces and must be different from the original name




Step 7
  1. Open the site string-hex-converter ---> https://codebeautify.org/string-hex-converter
  2. Paste the copied name in the top box




Step 8
  1. Copy the "Encoded string" from the bottom box




Step 9
  1. Change the copied profile with the just generated "Encoded string"




Step10
  1. Start ATS or ETS2 and know you new proflile will be in the dropdown list
Note: Do not forget to set the check "Use Steam Profile" back on.

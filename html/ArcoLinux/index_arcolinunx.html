ArcoLinux is an Arch Linux distribution.

They would like to gradually take you deeper and deeper into the Arch Linux world.
You will get full freedom to change the distro any way you want.
It is easy if someone explains it to you. 2000+ Erik Dubois Youtube tutorials and counting.

The motto of ArcoLinux : Learn, enjoy and have fun.


Phase 1 : install ArcoLinuxL and learn to use Xfce. You can install Openbox and i3wm to follow our learning path.

Phase 2 : install ArcoLinuxD and learn to install just Xfce or Openbox or i3 and use it

Phase 3 : install ArcoLinuxD and learn to install Awesome, Budgie, Cinnamon, Gnome, Mate, ... and learn how to use it

Phase 4 : Build your own iso on ArcoLinux or ArcoLinuxD iso - personal touch

Phase 5 : install Arch Linux and follow the Arch wiki installation guide

Phase 6 : build your own iso from scratch - how do we even start making a brand new iso/distribution

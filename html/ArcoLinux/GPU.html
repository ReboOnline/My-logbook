Check your GPU

Check which GPU your PC is running
The command below will display information about the graphics card(s).
It uses lspci to list all PCI devices and grep to filter the output for the VGA controller.
The -A 2 option tells grep to include two lines of context after the match, which will show additional information about the graphics card.

Copy and paste the following command in the terminal:
lspci -k | grep -A 2 -i "VGA"

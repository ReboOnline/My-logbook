How to automount a harddisk after booting up in ArcoLinux

Go to Disks and selct the seccond HDD or SSD to automount
Then the icon with th e tree wheels and go select Edit Mount Options
set set User Sessions Defaults off
and do not change the rest.
Klik on OK and reboot (reboot can later to).
Now the disk will mount on autostart

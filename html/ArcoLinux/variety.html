Variety settings

  1. Open Variety
  2. Set in General when to change the wallpapers
  3. Klik on "Add" to add a custom list.
  4. For example select Wallhaven.cc and enter keywords like toplist or fall
  5. Then set your favorite wallpapers to the right folder by selecting the right folder

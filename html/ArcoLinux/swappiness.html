Reduce swappiness

To reduce swappiness in ArcoLinux,
you can follow these steps:
  1. Open a terminal by pressing Ctrl+Alt+T or via the applications menu.

  2. To check your swappiness copy and paste the following command in the terminal:
    cat /proc/sys/vm/swappiness

  3. Edit the sysctl.conf file using a text editor with administrative privileges. For example, I am using the following command to open the file with the Nano text editor:
    sudo nano /etc/sysctl.conf

  4. Within the sysctl.conf file, add the following line at the end:
    vm.swappiness=10

  5. Save the changes by pressing Ctrl+O, then exit Nano by pressing Ctrl+X.

  6. Apply the swappiness value by copy and paste the following command in the terminal:
    sudo sysctl -p

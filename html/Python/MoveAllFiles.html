Move all files in a directory to another directory

To move all files in a directory to another directory using Python, you can use the shutil module's move function.
Here is an example of how you can do this:


  import shutil
  import os
  
  # Set the source and destination directories
  source_dir = '/path/to/source/directory'
  dest_dir = '/path/to/destination/directory'
  

  # Get a list of all the files in the source directory
  files = os.listdir(source_dir)
  
  # Iterate over the list of files
  for file in files:
       # Construct the full file paths
      source_path = os.path.join(source_dir, file)
      dest_path = os.path.join(dest_dir, file)
      # Use shutil.move to move the file from the source to the destination
      shutil.move(source_path, dest_path)
This code will move all the files in the source_dir to the dest_dir, preserving the directory structure. If you want to flatten the directory structure and move all the files directly into the destination directory, you can modify the dest_path as follows:

dest_path = os.path.join(dest_dir, os.path.basename(file))

This will move all the files directly into the dest_dir, rather than preserving the original directory structure.

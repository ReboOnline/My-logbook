Setting Up Python Development in VS Code on Linux Mint

This guide provides a step-by-step approach to setting up a Python development environment using Visual Studio Code (Code) on Linux Mint.

1. Preparation:

  1. Open your terminal and add the Microsoft GPG key and repository:
      wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg sudo install -o root -g root -m 644 packages.microsoft.gpg /usr/share/keyrings/ sudo sh -c 'echo "deb [arch=amd64 signed-by=/usr/share/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/vscode stable main" > /etc/apt/sources.list.d/vscode.list'
  1. Update your package sources and install VS Code:
      sudo apt update sudo apt install code

Install Python:

  1. Verify if Python is installed:
      python3 --version
  1. If not installed, add it with:
      sudo apt install python3 python3-pip

2. Setting Up VS Code for Python

Essential Extensions:

  1. Open VS Code.
  1. Go to the Extensions Marketplace (Ctrl+Shift+X) and install these extensions:
    • Python (Microsoft): For IntelliSense, debugging, and linting.
    • Pylance (Microsoft): Fast and enhanced code analysis.
    • Jupyter (Microsoft): For interactive Python notebooks.
    • GitLens (Optional, for version control).
    • Prettier (Optional, for consistent code formatting).

Configure Python Interpreter:

    • Press Ctrl+Shift+P → Type Python: Select Interpreter.
    • Choose the desired Python version (usually /usr/bin/python3).

Configure Linters and Formatters:

  1. Add the following settings to your settings.json:
      { "python.linting.enabled": true, "python.linting.pylintEnabled": true, "python.formatting.provider": "black", "editor.formatOnSave": true }
  • Install Black and Pylint in your environment:
      pip3 install pylint black

3. Optional Settings

Virtual Environment:

  1. Create a virtual environment:
      python3 -m venv .venv
  1. Activate it:
      source .venv/bin/activate
  1. Set VS Code to use the virtual environment (Ctrl+Shift+P → Python: Select Interpreter).

Reboot Linux via the terminal 1:

A simple bash script that can be used to update Linux:
 First make a file with the name "reboot"
#!/bin/bash echo "" sleep 2 echo "The computer wil Reboot now" sleep 1 echo "Until we meet again" sleep 1 clear ##-----clear the terminal-----## reboot
Make the script executable:
To make a bash script executable, you need to set the appropriate permissions on the file.
You can do this using the chmod command.
This will give the file execute permission for the owner of the file. chmod +x reboot
Copy the file to the /usr/local/bin folder:
This way you can use the command "up" annywhere in a terminal. Use the the code below in the terminal.
sudo cp reboot /usr/local/bin

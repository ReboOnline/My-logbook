Update Linux Mint via the terminal:

A simple bash script that can be used to update Linux Mint:
 First make a file with the name "up" 


  #!/bin/bash

  # this will clear the terminal
  Clear

  # update package list
  sudo apt-get update

  # echo "" will write a empty line in the terminal
  echo ""

  # sleep 2 will wait 2 seconds before the next command will start
  sleep 2

  # upgrade all installed packages
  sudo apt-get upgrade

  echo ""

  sleep 2

  # clean up obsolete packages
  sudo apt-get autoremove

Make the script executable:
To make a bash script executable, you need to set the appropriate permissions on the file.
You can do this using the chmod command.
This will give the file execute permission for the owner of the file. 


  chmod +x up

Copy the file to the /usr/local/bin folder:
This way you can use the command "up" annywhere in a terminal. Use the the code below in the terminal. 


sudo cp up /usr/local/bin

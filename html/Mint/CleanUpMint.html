How to cleanup Linux Mint via terminal

There are several ways to clean up a Linux Mint system via the terminal:
  1. Remove unnecessary or unused packages: You can use the apt-get command to remove unnecessary or unused packages.
    2. sudo apt-get autoremove

  2. Clear the package cache: The package manager keeps copies of all downloaded packages in a cache, which can take up a lot of disk space over time. You can clear the cache by running:
    3. sudo apt-get clean

  3. Remove old kernels: Linux Mint keeps multiple kernels as a fallback in case something goes wrong with a new kernel update. However, these old kernels can take up a lot of space. You can remove old kernels using the "purge-old-kernels" script:
    4. sudo apt-get purge-old-kernels

  4. Remove temporary files: Temporary files, such as those created by the system or by applications, can accumulate over time and take up a lot of space. You can use the tmpreaper utility to remove these files:
    5. sudo tmpreaper

  5. Compact the filesystem: If you have a lot of small files on your system, they can cause the filesystem to become fragmented, which can lead to slower performance. You can use the e4defrag utility to compact the filesystem:
    6. sudo e4defrag /

  6. Remove log files: Log files can also accumulate over time and take up a lot of space. You can use the logrotate utility to remove old log files:
    7. sudo logrotate -f /etc/logrotate.conf

Here is a script that combines the examples I provided above into a single script:

#!/bin/bash # Remove unnecessary or unused packages sudo apt-get autoremove # Clear the package cache sudo apt-get clean # Remove old kernels sudo apt-get purge-old-kernels # Remove temporary files sudo tmpreaper # Compact the filesystem sudo e4defrag / # Remove log files sudo logrotate -f /etc/logrotate.conf
  1. To use this script name it cleanup.sh save it and make it executable with the following command:
    2. chmod +x cleanup.sh

  2. To run the script open an terminal in the same direcrory where you have saved the file and type:
    3. ./cleanup.sh

  3. To run the script from anyware in the system in the terminal you have to copy the file to the right direcrory with the following command:
    4. sudo cp cleanup.sh /usr/local/bin
    Note: Make sure you to open the terminal in de same direcrory where the file is saved

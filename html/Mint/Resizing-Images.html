Resizing Images

Resizing Images with the convert command can quickly resize an image.
Use the following command in the terminal will the image will resize to 200 pixels in width and 100 pixels in height: convert *.png -resize 200x100 *.png

To resize all pictures in a folder in Linux, you can use the following bash script: 


  #!/bin/bash

  # Set the target directory
  TARGET_DIR="your/target/directory"
  
  # Set the target image width and height
  WIDTH="your-desired-width"
  HEIGHT="your-desired-height"
  
  # Loop through all files in the target directory
  for file in $TARGET_DIR/*
  do
    # Check if the file is a JPEG image
    if [[ $file == *.jpg || $file == *.jpeg ]]
    then
      # Use the 'convert' command to resize the image
      convert $file -resize $WIDTHx$HEIGHT $file
    fi
  done


In this script, you will need to set the TARGET_DIR variable to the path of the directory containing the images you want to resize. You will also need to set the WIDTH and HEIGHT variables to the desired width and height of the resized images. The script will then loop through all files in the target directory and use the convert command to resize any JPEG images it finds. It's important to note that this script will overwrite the original images, so be sure to make a backup of your images before running this script.

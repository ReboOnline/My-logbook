reboot-and-shutdown.sh

a Bash script that will reboot Linux Mint and then shut down after the reboot:

#!/bin/bash

# Reboot the system
sudo reboot
  
# Wait for the system to come back online
sleep 60
  
# Shut down the system
sudo shutdown -h now

  1. To use this script name it reboot-and-shutdown.sh, save it and make it executable with the following command:
    2. chmod +x cleanup.sh

  2. To run the script open an terminal in the same direcrory where you have saved the file and type:
    3. ./reboot-and-shutdown.sh

  3. To run the script from anyware in the system in the terminal you have to copy the file to the right direcrory with the following command:
    4. sudo cp reboot-and-shutdown.sh /usr/local/bin
    Note: Make sure you to open the terminal in de same direcrory where the file is saved

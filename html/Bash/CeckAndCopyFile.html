To check if a file is in a specific folder and notify
You can use the test command with the -f option, which checks if a file exists and is a regular file, and the echo command to print a message to the terminal. For example: 

  if test -f /path/to/folder/filename; then
      echo "File exists in folder"
  else
      echo "File does not exist in folder"
  fi

To copy a file to a specific folder:
You can use the cp command, for example: 

  cp /path/to/source/file /path/to/destination/folder

To notify when the file has been copied:
You can use the echo command to print a message to the terminal. For example: 

    echo "File copied successfully"
If an error occurs while trying to copy the file
You can use the echo command to print an error message. For example: 

    echo "Error: Could not copy file" >&2
combines these commands
Here is an example script that combines these commands to check if a file is in a specific folder and notify, copy it to that folder, and notify when done or if an error occurred: 

  #!/bin/bash
  
  # Check if file exists in folder
  if test -f /path/to/folder/filename; then
      echo "File exists in folder"
  else
      echo "File does not exist in folder"
      exit 1
  fi
  
  # Copy file to destination folder
  cp /path/to/source/file /path/to/destination/folder
  
  # Check if copy was successful
  if [ $? -eq 0 ]; then
      # Print success message
      echo "File copied successfully"
  else
      # Print error message
      echo "Error: Could not copy file" >&2
  fi
This script can be executed by running ./script.sh in the terminal, where script.sh is the name of the script file.

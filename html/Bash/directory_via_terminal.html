The directory via the terminal

To display the current working directory in a terminal
You can use the pwd command:
$ pwd
Will put out "/home/user/documents"


To change the current working directory in a terminal:
For example, to change to the documents directory located in the home directory,
you would run the following command:
$ cd documents


Change to a directory that has a space in its name:
If you want to change to a directory that has a space in its name,
you can enclose the directory name in quotation marks or escape the space with a backslash " \ ":
$ cd "My Documents"
wil be:
$ cd My\ Documents
